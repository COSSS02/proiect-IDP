# Changelog

## [1.7.0] - 2026-04-26
### Added
- Utilitarul de gestionare a bazelor de date Adminer.
- Certificate HTTPS pentru comunicare securizata atat pentru server, cat si pentru client.

## [1.6.0] - 2026-04-18
### Added
- Configuratie Docker Swarm pentru orchestrarea implementarii.

## [1.5.0] - 2026-04-13
### Added
- Scripturi pentru popularea bazei de date.

## [1.4.0] - 2026-04-10
### Added
- Componente (navbar, sidebar, paginatie, panou pentru afisarea produselor) si pagini (pagina cu toate produsele, pagina cu detaliile unui produs, pagina cu cosul de cumparaturi) necesare pentru construirea front-endului.

## [1.3.0] - 2026-04-05
### Added
- Implementare completa pentru backend, incluzand controllere, modele si rute pentru utilizatori, adrese, cos de cumparaturi, categorii, produse, atribute, comnenzi si wishlist.
- Implementare autentificare folosind keycloak.
- Configuratie initiala pentru client folosind Vite.
- Proxy-uri API pentru conexiunea dintre frontend si backend.

## [1.2.0] - 2026-04-01
### Added
- Initializarea serverului backend si configurarea de baza a serviciului web si a comunicarii cu baza de date.

## [1.1.0] - 2026-03-30
### Added
- Script de initializare MariaDB pentru crearea structurii de baza a bazei de date.

## [1.0.0] - 2026-03-29
### Added
- Structura initiala a proiectului.
