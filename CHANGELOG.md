- Github Workflow pentru actualizarea automata a imaginilor din dockerhub.

- Portainer pentru a vizualiza stackul din interfata grafica.

### Modified
- Stack deployment pentru a folosi un nod manager si doi workeri.

- Separare servicii intre retele diferite.

### Added
- Monitorizare prin Prometheus si dashboard de vizualizare prin Grafana.

### Added
- Kong API Gateway prin care se acceseaza toate celelalte servicii.

### Added
- Utilitarul de gestionare a bazelor de date Adminer.

## [1.6.0] - 2026-04-18
### Added

## [1.5.0] - 2026-04-13
### Added
- Scripturi pentru popularea bazei de date.

## [1.4.0] - 2026-04-10
### Added
- Componente (navbar, sidebar, paginatie, panou pentru afisarea produselor) si pagini (pagina cu toate produsele, pagina cu detaliile unui produs, pagina cu cosul de cumparaturi) necesare pentru construirea front-endului.

## [1.3.0] - 2026-04-05
### Added
- Implementare completa pentru backend, incluzand controllere, modele si rute pentru utilizatori, adrese, cos de cumparaturi, categorii, produse, atribute, comnenzi si wishlist.

- Configuratie initiala pentru client folosind Vite.
- Proxy-uri API pentru conexiunea dintre frontend si backend.
